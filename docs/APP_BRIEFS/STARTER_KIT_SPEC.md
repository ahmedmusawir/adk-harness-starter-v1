## CONTEXT

This repo is a production ADK multi-agent bundle deployed on Google Cloud Run. It contains 5 agents, an N8N gateway integration, a FastAPI wrapper layer, and Supabase session persistence.

We are converting it into a **starter kit** for building Phase 1 POC agents. The goal is to strip out what we don't need, keep what's proven, and add stubs for new shared utilities.

**Rule:** Do NOT modify any working code in files that are being kept. Only remove files/folders and add new stubs.
